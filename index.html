Buttons

There are 4 types of buttons:

  • Filled
  • Outlined
  • Tonal
  • Normal

Aditionally, you can add elevated and/or main. Elevated buttons seem lifted off the ground, and main buttons are supposed to be the only button or the main button on the page.

Buttons with Variants

Usage

You can use the buttons in your HTML like this:

    <button filled>Filled</button>

<button outlined>Outlined</button>

<button tonal>Tonal</button>

<button>Normal</button>

<button main>Main</button>

<button elevated>Elevated</button>

<button main tonal>Main Tonal</button>

<button elevated outlined>Elevated Outlined</button>

Button Groups

Button groups are used to group related buttons together. They can be used to create a toolbar or a set of related actions. There are two types of button groups: selectable and connected.


Selectable

Selectable button groups are used to group selectable buttons that are related.

Connected

Connected button groups are used to group buttons that are related and should be connected together. They are used when the user needs to select one option from a set of options.



Usage

You can use the button groups in your HTML like this:

    <!-- Normal button group -->

<button-group>
  <button>Button 1</button>
  <button selected>Button 2</button>
  <button>Button 3</button>
  <button>Button 4</button>
  <button selected>Button 5</button>
</button-group>


<!-- Normal button group with rounded corners -->

<button-group rounded>
  <button selected>Button 1</button>
  <button>Button 2</button>
  <button selected>Button 3</button>
  <button>Button 4</button>
  <button>Button 5</button>
</button-group>


<!-- Connected button group -->

<button-group connected>
  <button>Button 1</button>
  <button>Button 2</button>
  <button>Button 3</button>
  <button selected>Button 4</button>
  <button>Button 5</button>
</button-group>
